# TS + Rollup Library Template

Clone:

```bash
git clone https://github.com/danielFHcode/ts-rollup-library-template.git
```

Get rid of the evidence:
```bash
rm -rf .git
```

 Update the package.json:
 ```jsonc
    "name": "package-name", // <--
    "version": "1.0.0",
    "main": "dist/index.mjs",
    "types": "dist/index.d.ts",
    "type": "module",
```

And get started coding!
